# ChamRisk
Manage software risk on OpenSUSE Tumbleweed
Developed by Richard Northwood
## Project status

ChamRisk is a shelved prototype / early build shared for review and possible collaboration.

It does not currently meet the standard required for release. The main issues are UI/layout instability, framework suitability, and packaging/distribution constraints.

The codebase is being published for transparency and possible salvage, not as an example of polished production work.
