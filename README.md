# ChamRisk
Manage software risk on OpenSUSE Tumbleweed
Developed by Richard Northwood
## Project status

ChamRisk is a shelved prototype / early build shared for review and possible collaboration.

It does not currently meet the standard required for release. The main issues are UI/layout instability, framework suitability, packaging/distribution constraints, and accumulated prototype debt.

The codebase likely contains partial implementations, redundant paths, and unfinished experiments. It is being published for transparency and possible salvage, not as an example of polished production work.
